# 直男翻译官 · 情侣双人测评

> 9 题测出他的浪漫段位，看看你们的默契度有多高。  
> A couple's compatibility test, with a twist.

🔗 **在线体验**：https://shzzzayys.github.io/zhinan-test/

---

## ✨ 这是什么

「直男翻译官」是一款情侣向的轻量化测评 H5。  
不需要下载、不需要注册、不需要后端服务器，**打开就能玩**。

---

## 🎯 核心玩法

### 玩法 A · 情侣双人版

```
女生答 9 题 → 把手机递给男友 → 男友答 9 题 → 立刻出对比报告
```

测出男友的「直男段位」+ 两人的「默契度」。

### 玩法 B · 单人快测

```
1 个人答 9 题 → 30 秒看到结果
```

适合一个人想偷偷测男友，或者朋友间互相调侃用。

## 🧠 设计亮点

- **30 题题库 · 每次随机 9 题**：每次玩都不一样，玩 10 次都不重样
- **六维人物画像**：情绪感知 / 表达温度 / 细节观察 / 浪漫指数 / 冲突处理 / 行动力
- **同手机轮流答**：双人模式不需要发链接，递手机就行
- **精致海报分享**：可生成 9:16 海报，长按保存到相册
- **零依赖**：纯 HTML/CSS/JS 单文件，无后端，无数据库，无第三方 SDK
- **隐私友好**：所有数据只在你的浏览器内存中，不上传任何服务器

## 🚀 本地运行

不需要任何依赖，下载 `index.html` 双击即可在浏览器打开。

或者用任意 HTTP 服务：

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .

# 然后访问 http://localhost:8000
```

## 🛠️ 技术栈

- **HTML5 + CSS3 + 原生 JavaScript**
- **字体**：Playfair Display（标题）、Noto Serif SC（中文宋体）、Noto Sans SC（中文正文）
- **设计系统**：奶油白 (#FAF6F0) + 暖橘 (#FF6B35) + 墨绿 (#2D4A3E)
- **托管**：GitHub Pages
